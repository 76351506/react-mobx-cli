  - 目录规范


## 将本地项目提交到远程
   - git init 初始化项目
   - git remote add origin http://.... 添加远程服务器地址
   - git pull origin master --allow-unrelated-histories 拉取远程代码并合并，可能会产生冲突
   - 解决冲突
   - git add .
   - git commit -m ''
   - git push --set-upstream origin master 强推
